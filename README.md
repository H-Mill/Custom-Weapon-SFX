# Squeaky Toy

A RuneLite plugin that plays a squeaky toy sound effect on configurable player actions.

## Features

- **Player Attacks** — squeak on outgoing hits: all attacks or zero-damage only
- **Attacks on Player** — squeak on incoming hits: all attacks or zero-damage only
- **Volume** slider (0–100)
- Each option is independently configurable; all default to disabled except Player Attacks (All attacks)

## Config

| Setting | Description | Default |
|---|---|---|
| Volume | Master volume of the squeak (0–100) | 75 |
| Player Attacks | When to squeak for your outgoing attacks | On zero |
| Attacks on Player | When to squeak for attacks received by you | Disabled |
